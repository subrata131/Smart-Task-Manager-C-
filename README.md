# Smart Task Manager (C)

A simple **command-line Smart Task Manager** built using the **C programming language**.  
This project allows users to manage tasks efficiently by assigning priorities and tracking their completion status.

The system simulates how tasks can be organized and executed based on **priority levels**, similar to basic scheduling concepts used in operating systems.

---

## Features

- Add new tasks with name, priority, and deadline
- Assign task priority (High / Medium / Low)
- View all tasks
- Sort tasks by priority
- Mark tasks as completed
- Menu-driven interface for easy interaction
